# E-Commerce Performance & Customer Satisfaction Analytics (Olist)
# Project Overview
This project analyzes the Olist Brazilian E-Commerce dataset to uncover key business insights related to revenue growth, customer satisfaction, seller performance, and operational efficiency.

The objective is to simulate the role of a Data Analyst working for an e-commerce marketplace and provide actionable recommendations based on data-driven analysis.

The project covers the complete analytics lifecycle:
* Data Understanding
* Data Cleaning & Validation
* SQL Analysis
* Exploratory Data Analysis
* Statistical Analysis
* Business Insight Generation
* Power BI Dashboard Development
* Executive Recommendations

# Business Problem
* Which product categories generate the most revenue?
* Which sellers contribute the most to marketplace performance?
* What factors drive customer satisfaction?
* How do delivery delays impact review scores?
* Which regions contribute the most sales?
* What opportunities exist to improve customer experience and business performance?

# Dataset
**Dataset:** Olist Brazilian E-Commerce Public Dataset(kaggle)
The dataset contains information from a Brazilian e-commerce marketplace, including:
* customers
* orders
* order_items
* products
* sellers
* order_payments
* order_reviews
* geolocation

The data spans multiple business functions and provides a realistic environment for end-to-end business analytics.
